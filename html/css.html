    CSS:

    Site pessoal para treinar HTML5 e CSS e expor meus conhecimentos, com minhas considerações sobre o código, erros e melhorias!

    Comentários

    1. Considerações iniciais: A página principal do site foi meu primeiro projeto em CSS, revisei o código diversas vezes e tentei incorporar o máximo de técnicas para aprender diferentes funcionalidades como "flex box" e o "grid".

    2. Erros: Tentei ao máximo fazer um site responsivo que tivesse aparencia boa em qualquer plataforma, para atingir esse objetivo utilizei um layout simples. Infelizmente a estética da página inicial ainda deixa a desejar

    3. Melhorias: Embora tenha conseguido utilizar tanto o grid quanto o flex, utilizei ambos de forma bem simplificada, na versão futura do site irei tentar ambas as funcionalidades de forma mais complexa, com o intuito de melhorar a estética.

    Comentários

    1. Considerações: Utilizei varias ferramentas para deixar o site responsivo, apesar da estética ainda precisar de melhorias, o site está funcional.

    2. Erros: Apesar de ter um bom entendimento das funcionalidades "margins" e "paddins", ainda não tenho o total dominio dos diferentes objetos. O que acaba gerando algumas redundâncias e má utilização de ambas as funções.

    3. Melhorias: Preciso aprimorar meu entendimento dos diferentes "objetos" e "boxes", acredito que essa habilidade vira com mais prática e estudos.

    Comentários

    1. Considerações: Adicionei diferentes cores ao "hover" dos botões, para praticar a animação, porem estéticamente os mesmos deixaram a desejar.

    2. Erros: Acredito que o maior erro, foi o visual, estéticamente o site e os botões precisam de melhorias, mais especificamente, não consegui separar a imagem dos botões do texto.

    3. Melhorias: Preciso melhorar a estética vísual do site e criar animações mais agradáveis.

    Comentários

    1. Para criar as diferentes cores da animção do "hover" do mouse sobre o botão utilei um id específico para cada botão, não tenho certeza se essa é a melhor opçãos, preciso me aprofundar mais tanto no HTML quanto no CSS.

    Comentários

    1. Considerações: Percebi após a revisão do código CSS que utilizei muito as funções "margins" e "padding" para fazer algumas "gambiarras" de itens cujo o comportamento eu não entendi completamente, isso é uma boa indicação de onde preciso focar para melhorar.

    Considerações Finais:

    O site tem vários problemas, tanto estéticos quanto técnicos, porem como foi meu primeiro projeto, acredito que tenha aprendido muito e ganho um pouco de experiência "hands on", o que foi muito valioso.

    Como estou aprendendo por cursos gratuitos disponibilizados pela internet, projetos como este site são importantes para deixar claro os pontos fortes e fracos desse aprendizado. Estou atualmente trabalhando na versão 2.0 do site e estudando JavaScript para melhora-lo.
