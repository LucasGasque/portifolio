    Projeto de controle de estoque em python,
    para treinar a linguagem, com minhas considerações
    sobre o código, erros e melhorias!

    Comentários

    1. Considerações iniciais: utilizei a função "print" para criar a interface "gráfica" e um loop "While" para criar a navegação.

    2. Uso do loop "WHILE" talvez não seja a melhor opção, porem ainda não domino a função def, que, acho, seria ideal para estabelecer a "navegação".

    3. Melhorias: Preciso aprender e adicionar uma interface gráfica ao programa.

    Comentários

    1. Considerações: Utilizei 3 dicionarios para fazer o controle, 1 para cadastro de itens, 1 para quantidade e 1 para vendas.

    2. Erros: Uso exagerado da função "If, Else", a não consolidação dos 3 dicionarios em um só.

    3. Melhorias: Preciso aprender e adicionar um banco de dados ao programa, os dados ficam salvos na memória volatil e se perdem ao fechar o terminal.

    Comentários

    1. Considerações: Criei uma condição para o cadastro de quantidades de itens, para verificar se o item já possuia cadastro de preço.

    2. Erros: Considerei a retirada de itens do estoque como sinonimo de vendas, o que é inadequado, nem toda retiada é uma venda.

    3. Melhorias: Adicionar uma correção ou retirada do estoque que não seja considerada uma venda.

    Comentários

    1. Considerações: Criei a condição de "Opção invalida" para o programa não fechar caso receba um input invalido e a opção do input "encerrar" para finalizar o programa.

    Considerações Finais:

    O objetivo do programa era treinar a linguagem Python, e não criar algo funcional, porem ao pesquisar mais sobre a correta aplicação de determinadas funcionalidades, tive um bom direcionamento do que preciso aprender no futuro: uma ferramenta de interface gráfica, um sistema de banco de dados e possivelmente outras ferramentas.

    Também percebi minhas limitações na linguagem, mais especificamente quanto a função dicionarios e, principalmente, quanto a função "def" e a criação de classes. Ambas serão meu foco em projetos futuros!
